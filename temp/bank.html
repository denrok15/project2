Решить при всех значениях параметра уравнение x − a = 2
Данное уравнение можно переписать в виде x − a = 4 при условии, x − a ⩾ 0 . Следовательно, получаем:

{ x = 4 + a x ⩾ a
Если 4 + a ⩾ a , то корень x = 4 + a удовлетворяет условию x ⩾ a , то есть система имеет единственное решение
x = 4 + a
Если 4 + a < a ,то корень x = 4 + a не удовлетворяет условию x ⩾ a , то есть система не имеет решений.

Ответ:
x = 4 + a при всех a ∈ R
