Прямая пропорциональность

Определение

  • Функция прямой пропорциональности имеет вид y = kx + b, где k и b - коэффициенты.
  • График функции - прямая линия в плоскости (x, y).
  • Нули функции - точка (0, b).
  • Прямая проходит через начало координат, если b = 0.
  • Функция является нечетной, если k нечетное число; и четной, если k четное.

Свойства

  • Область определения - все действительные числа (x принимает любые значения).
  • Область значений - все действительные числа (y принимает любые значения).
  • Промежутки возрастания - при k > 0, функция возрастает; при k < 0, функция убывает.
  • Промежутки убывания - при k > 0, функция убывает; при k < 0, функция возрастает.
