Математика: Свойства степени и логарифма

Свойства степени

1. Произведение степеней с одинаковым основанием

Если a - положительное число, то am * an = am+n.

2. Деление степеней с одинаковым основанием

Если a - положительное число, то am / an = am-n.

3. Возведение в степень степени

Если a - положительное число, то (am)n = am * n.

4. Степень числа 1

Любое число, возведенное в степень 0, равно 1: a0 = 1 (при условии a ≠ 0).

5. Степень с отрицательным показателем

Если a - положительное число и n - отрицательное число, то an = 1/a|n|.

6. Степень равна 1

Любое число, возведенное в степень 1, равно самому себе: a1 = a.

7. Возведение в степень числителя и знаменателя

Если a и b - положительные числа, то (a/b)n = an / bn.

Свойства логарифма

1. Определение логарифма

Логарифм числа b по основанию a, обозначаемый loga(b), равен степени, в которую нужно возвести a, чтобы получить b.

2. Свойство логарифма от произведения

loga(xy) = loga(x) + loga(y).

3. Свойство логарифма от частного

loga(x/y) = loga(x) - loga(y).

4. Свойство логарифма от степени

loga(xn) = n * loga(x).

5. Свойство логарифма от числа 1

loga(1) = 0, независимо от основания логарифма a.

6. Свойство логарифма от основания

loga(a) = 1, где a - положительное число и a ≠ 1.

7. Свойство смены основания логарифма

loga(b) = logc(b) / logc(a), где a, b, и c - положительные числа, и c ≠ 1.

Свойства корня

1. Определение корня

Корень степени n из числа a, обозначаемый как √n(a), равен такому положительному числу x, что xn = a.

2. Свойство корня от произведения

√n(ab) = √n(a) * √n(b).

3. Свойство корня от частного

√n(a/b) = √n(a) / √n(b), при условии, что √n(b) ≠ 0.

4. Свойство корня от степени

√n(am) = am/n, где m - целое число, а n - натуральное число, и m/n - дробь в простейшем виде.

5. Свойство корня от числа 1

√n(1) = 1, независимо от значения n.

6. Свойство корня от отрицательного числа

Корень неопределенного порядка из отрицательного числа не является вещественным числом (в общем случае).

7. Свойство корня от нуля

√n(0) = 0, независимо от значения n.

Формулы сокращенного умножения

1. Квадрат суммы двух чисел

(a + b)2 = a2 + 2ab + b2.

2. Квадрат разности двух чисел

(a - b)2 = a2 - 2ab + b2.

3. Разность квадратов

a2 - b2 = (a + b)(a - b).

4. Куб суммы двух чисел

(a + b)3 = a3 + 3a2b + 3ab2 + b3.

5. Куб разности двух чисел

(a - b)3 = a3 - 3a2b + 3ab2 - b3.

6. Сумма кубов двух чисел

a3 + b3 = (a + b)(a2 - ab + b2).

7. Разность кубов двух чисел

a3 - b3 = (a - b)(a2 + ab + b2).

Основные формулы из тригонометрии

Пусть a - угол в радианах:

  • Sin(a) = противоположная сторона / гипотенуза
  • Cos(a) = прилежащая сторона / гипотенуза
  • Tan(a) = противоположная сторона / прилежащая сторона
  • Cot(a) = прилежащая сторона / противоположная сторона
  • Sec(a) = гипотенуза / прилежащая сторона
  • Csc(a) = гипотенуза / противоположная сторона

Дополнительные формулы:

  • Sin2(a) + Cos2(a) = 1
  • Tan(a) = Sin(a) / Cos(a)
  • Cot(a) = 1 / Tan(a)
  • Sin(2a) = 2 * Sin(a) * Cos(a)
  • Cos(2a) = Cos2(a) - Sin2(a)
  • Tan(a) * Cot(a) = 1
  • 1 + Tan2(a) = Sec2(a)
  • 1 + Cot2(a) = Csc2(a)
  • Sin(a ± b) = Sin(a) * Cos(b) ± Cos(a) * Sin(b)
  • Cos(a ± b) = Cos(a) * Cos(b) ∓ Sin(a) * Sin(b)
  • Tan(a ± b) = (Tan(a) ± Tan(b)) / (1 ∓ Tan(a) * Tan(b))
  • Cot(a ± b) = 1 / Tan(a ± b)
  • Sin(2a) = 2 * Sin(a) * Cos(a)
  • Cos(2a) = Cos2(a) - Sin2(a)
  • Tan(2a) = (2 * Tan(a)) / (1 - Tan2(a))
  • Cot(2a) = (Cot2(a) - 1) / (2 * Cot(a))
  • Sin2(a) = (1 - Cos(2a)) / 2
  • Cos2(a) = (1 + Cos(2a)) / 2
  • Tan2(a) = (1 - Cos(2a)) / (1 + Cos(2a))
  • Cot2(a) = (1 + Cos(2a)) / (1 - Cos(2a))
